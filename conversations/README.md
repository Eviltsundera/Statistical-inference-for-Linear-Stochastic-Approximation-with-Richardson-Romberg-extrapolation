# Math Conversations

This directory stores Markdown answers to user math questions.

Use one file per topic or tightly connected thread:

```text
YYYY-MM-DD_<short-topic>.md
```

Each note should be readable without the chat transcript. Include:

- the user's question or task;
- notation and assumptions needed for the answer;
- the main derivation or explanation;
- open gaps, checks, or next steps.

If a follow-up continues the same mathematical thread, update the existing note
instead of creating a near-duplicate file.
